# 感染管理マニュアル スマホ利用案内

感染管理マニュアルは、iPhone / Android のブラウザから利用できます。

## 利用方法

1. 配布されたURLを開きます。
2. 必要な項目を検索、または章から選んで開きます。
3. よく使う場合は、ホーム画面に追加してください。

## iPhoneでホーム画面に追加する

1. SafariでマニュアルのURLを開きます。
2. 画面下の共有ボタンを押します。
3. 「ホーム画面に追加」を選びます。
4. 名前を確認して「追加」を押します。

## Androidでホーム画面に追加する

1. ChromeでマニュアルのURLを開きます。
2. 右上のメニューを押します。
3. 「ホーム画面に追加」または「アプリをインストール」を選びます。
4. 案内に従って追加します。

## 補足

- 通信環境によって、図表の初回表示に少し時間がかかることがあります。
- 内容確認が必要な場合は、各項目の「公式PDF」も参照してください。
